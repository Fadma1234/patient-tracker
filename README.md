### Patient Tracker
<img width="1920" alt="image" src="https://github.com/user-attachments/assets/69e9c06d-c27b-445d-838e-d34337fcda2d" />

Patient Tracker is a simple patient tracking web application. It allows patients to log in and submit the exercise they have done, and receive reminders to do them. This application will allow the provider to give feedback to the patient as well.





### Tech Stack

- HTML

- CSS

- JavaScript

- Bootstrap

- Node.js

- Express.js

- MongoDB

- Cloudinary

- EJS
